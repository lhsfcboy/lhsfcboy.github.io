# Windows 10 操作系统的配置


## 操作系统基础设置
### 更改计算机名称
计算机名称会被用于蓝牙设备等.
- Settings / System / About /  Device specifications / Rename this PC
- Restart to apply

### Clear/Diable Windows 10 Quick Access History
- Click Start and type: file explorer options
- Privacy section
