# 真人CS / 生存游戏 / BB弹对战

## 买过的枪

- MP5 PWD
  - 优点:
    - 枪身短, 枪托可以翻折, 收纳方便
    - 有前握把, 持枪舒服
  - 缺点:
    - 威力太小, 直线射程近
    - 装配瞄准镜时, 需要垫高以避开原有的机械瞄具
  - 配件一览
    - 导轨和垫高
    - 红点瞄准镜
    - 多发弹夹
    - 背带  
    - 消音器以及配套螺丝换
      - 直线飞行距离略有上升, 声音略有下降
