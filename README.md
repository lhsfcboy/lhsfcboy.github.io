## 文件夹 & 文件名的命名规范
- 基本原则先英文后汉语
- 首字母大小写
  - 保持GitHub网站, Linux, Windows等主要平台下, 文件排序顺序一致
  - 同一层级的文件其首字母大小写保持一致
  - 同一层级的文件夹其首字母大小写保持一致
- 禁止空格
- 谨慎使用中文与英文以外的其他语言
  - 尤其避免中文和日语的汉字混用
- 谨慎使用不常见的字符
- 谨慎使用全大写的文件名
