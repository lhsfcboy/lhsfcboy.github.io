# Linux GCC

Linux中的GCC是一件很麻烦的事情, 轻易不要碰的好。
这也是Docker飞速发展的原因

## 常见发行版
- Fedora/RHEL/CentOS/Rocky
- Ubuntu

## 主要发行版的GCC版本

| Distribution     | Kernel | GCC   | GLIBC | Supported until |
|------------------|--------|-------|-------|-----------------|
| RHEL/CentOS 7.5  | 3.10   | 4.8.5 | 2.17  |                 |
| Ubuntu 18.04.1   | 4.15.0 | 7.3.0 | 2.27  | 2028-04         |
